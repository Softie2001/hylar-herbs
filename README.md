# Hylar Herbs & Body Wellness — Website

Natural herbal remedies brand based in Ede (Osun State) & Lagos, Nigeria.

## Structure
- `index.html` — the site
- `assets/` — product & logo images used by the site (must stay in this folder, same names)

## Deploy with GitHub Pages
Settings → Pages → Source: Deploy from branch → `main` → `/ (root)` → Save.
Site goes live at `https://<username>.github.io/<repo-name>/`.
